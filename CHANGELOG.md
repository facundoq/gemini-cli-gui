# Changelog

## [1.5.0] - 2025-07-15

### Added
- Support for Gemini 2.5 Pro model
- Model selection feature (selectable from settings screen)
- YOLO mode (support for --yolo flag)
- Model display in chat screen
- Improved image attachment functionality

### Changed
- Migration from Claude CLI UI to Gemini CLI UI
- Changed default model to auto
- Various UI branding updates
- Removal of unnecessary debug logs

### Fixed
- Chat input field alignment issues
- Layout breakage on initial load
- Improved session management

### Removed
- Claude-related files and components
- MCP Servers functionality (temporarily)
- Unnecessary debug log output
